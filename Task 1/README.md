# Iris Dataset Analysis

## Objective
The objective of this project is to perform Exploratory Data Analysis (EDA) on the Iris dataset using Python.

## Tools and Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Approach
1. Loaded the Iris dataset using Seaborn.
2. Explored dataset structure and columns.
3. Checked for missing values.
4. Created visualizations including:
   - Scatter Plot
   - Histogram
   - Box Plot
5. Analyzed patterns and differences between iris flower species.

## Results and Insights
- The dataset contains 150 flower records.
- Different iris species show noticeable differences in petal and sepal measurements.
- Visualizations helped understand the distribution and relationships between features.

## Conclusion
This project successfully demonstrated basic data analysis and visualization techniques using Python.
